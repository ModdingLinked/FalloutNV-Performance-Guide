DXVK - Overview

DXVK is a Direct3D 9/10/11 to Vulkan translation layer created mainly for Linux, but works unoffically on Windows.

In short, it makes games run using Vulkan API which has benefits of much lower overhead (especially when compared to D3D9), which most often grants increased performance in CPU bound games (Fallout games are in that group).

Warnings about DXVK you have to acknowledge:

  • DXVK does NOT guarantee higher performance on every system. In most cases it does, but there are some outliers.

  • DXVK does NOT work properly on laptops/ PCs with hybrid graphics. It always runs in Windowed mode with massive latency and broken framerate.

    It does work however on most Gen 9.5+ Intel graphics provided you have newest drivers installed.

  • DXVK can have some shader problems in games.
    In Fallout New Vegas known issues are:

    • Broken Transparency Multisampling (makes grass and transparent objects not anti-aliased)
    • (Rarely) Broken lights and artifacts

Due to DXVK's nature, it relies on shader cache, which is compiled during the first use of that shader.
Because of that you can experience increased short stutter at the beginning of your playthrough.
This is completely normal, and will not reoccur after compilation has finished.
DXVK-Async which will be used in this guide alleviates this problem thanks to making the compilation asynchronous.

DXVK - Installation

Normal

  1. Open DXVK-Async Github page and click on dxvk-async-1.10.1.tar.gz to download it.

  2. Extract d3d9.dll from x32 folder inside downloaded archive to your Fallout New Vegas root folder.

  3. Create dxvk.conf text file and write:

    dxvk.enableAsync = true

    This enables async patch in DXVK.

Alongside Special K

  1. Follow normal DXVK installation instructions.

  2. Rename d3d9.dll to dxvk.dll.

  3. Create d3d9.ini text file and write:

    [Import.DXVK]
    Architecture=Win32
    Role=d3d9
    When=Proxy
    Filename=dxvk.dll

    This makes Special K load DXVK before itself.

  4. Install Special K
