Introduction

Testing Methodology

Tests will vary in location and duration based on the type of mod and how it is expected to affect performance. For example asset replaces will most likely be tested statically. On the other hand, things related to asset streaming need to be tested with movement across cells.

All tests are conducted with this system configuration:

  • Native 1080p resolution.
  • Windows 11, latest public release at the time of testing.
    • Memory Integrity disabled.
  • Latest GPU driver at the time of testing.
  • 5800X3D, -30CO to let clock speeds reliably reach the intended amount according to the official specification (4.5Ghz).
  • 6700XT, stock.
  • 32GB at 3600Mbps, CL18-22-22-42 (XMP profile).
